# Data Folder

Place the dataset file here:

- `job_salary_prediction_dataset.csv` — 250,000 job records with salary labels

## Columns

| Column | Type | Description |
|---|---|---|
| job_title | string | Job role title |
| experience_years | int | Years of experience (0–20) |
| education_level | string | Highest education level |
| skills_count | int | Number of skills |
| industry | string | Industry sector |
| company_size | string | Company size category |
| location | string | Work location/city |
| remote_work | string | Remote / Hybrid / On-site |
| certifications | int | Number of certifications |
| salary | int | **Target** — Annual salary in USD |
